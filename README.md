## 🔍 Overview
*KeyMon* is an interactive, console-based keylogger designed for cybersecurity education and system monitoring. Built with Python and the `pynput` library, it features a bold, professional terminal interface that allows users to configure logging paths and control the monitoring process in real-time.

## ⚙️ How It Works
| Step | Action | Detail |
| :--- | :--- | :--- |
| **01** | **Hook** | Background Listener hooks into system input via `pynput`. |
| **02** | **Event** | Handlers differentiate between alphanumeric and special keys. |
| **03** | **Log** | Module appends data with high-precision timestamps. |
| **04** | **Safe** | Manual control allows for graceful listener shutdown. |
